1. Cuenta y acceso

Iniciar sesion

Introduce tu nombre de usuario y contrasenya en la pantalla de login. Tu sesion se mantiene activa mediante un token seguro (JWT).

Crear cuenta

  1. Pulsa "Crear cuenta" en la pantalla de login
  2. Introduce un nombre de usuario (minimo 3 caracteres)
  3. Introduce tu email
  4. Crea una contrasenya segura (minimo 8 caracteres, una mayuscula, un numero y un caracter especial)
  5. Revisa tu email para activar la cuenta

Verificacion en dos pasos (2FA)

Puedes activar 2FA desde tu perfil para mayor seguridad. Necesitaras una app como Google Authenticator o Authy.

  1. Ve a Perfil > Autenticacion 2FA
  2. Pulsa Activar 2FA
  3. Escanea el codigo QR con tu app de autenticacion (o introduce el codigo manual)
  4. Introduce el codigo de 6 digitos para confirmar
Una vez activado, cada vez que inicies sesion te pedira el codigo de tu app de autenticacion ademas de la contrasenya.

2. Dashboard (Inicio)

Es la pantalla principal que ves al entrar. Muestra un resumen rapido de tu actividad:

  • Dietas - Numero de dietas que tienes importadas/creadas
  • Planes - Numero de planes semanales activos
  • Ultimo peso - Tu ultima medicion de peso registrada
  • Registros - Total de registros de peso que has hecho

Cumplimiento semanal

Barra de progreso que muestra el porcentaje de comidas que has completado en tu plan semanal activo. Al 100% significa que has marcado todas las comidas de la semana.

Mis Dietas

Listado de todas tus dietas con botones de accion rapida:

  • Ver - Muestra el detalle de la dieta con todos los dias y comidas
  • Editar - Abre el editor para modificar la dieta
  • Plan - Crea un plan semanal a partir de esta dieta (empieza el lunes de la semana actual)
  • Export - Descarga la dieta en formato JSON
  • X - Elimina la dieta

Botones de importacion

  • PDF - Importar dieta desde archivo PDF
  • Imagen - Importar dieta desde foto/imagen
  • Dieta - Crear dieta manualmente
  • Plan - Crear plan semanal manual (sin dieta asociada)
  • Import - Importar dieta desde archivo JSON

3. Gestion de dietas

Las dietas son la base de tus planes semanales. Puedes crearlas de varias formas:

Importar desde PDF

  1. Pulsa el boton PDF en el dashboard
  2. Selecciona tu archivo PDF de dieta
  3. Pon un nombre descriptivo (ej: "Dieta Marzo 2026")
  4. El sistema extrae automaticamente los alimentos de cada dia y comida
El PDF debe tener formato de tabla con columnas por dia (DIA 1, DIA 2...) y filas por comida (Desayuno, Almuerzo, Comida, Merienda, Cena).

Importar desde imagen

  1. Pulsa el boton Imagen en el dashboard
  2. Selecciona una foto o captura de tu dieta (PNG, JPG, WEBP, etc.)
  3. Pon un nombre a la dieta
  4. El sistema analiza la imagen con inteligencia artificial y extrae la estructura

Crear dieta manual

  1. Pulsa el boton Dieta en el dashboard
  2. Pon un nombre a la dieta
  3. Navega entre los 7 dias (Lunes a Domingo) con las pestanyas
  4. Anade comidas: selecciona el tipo (Desayuno, Comida, Cena...) y pulsa + Comida
  5. Dentro de cada comida, anade alimentos con nombre, cantidad y busca las kcal
  6. Pulsa Guardar dieta cuando termines

Busqueda de alimentos y calorias

Al crear o editar una dieta manualmente, puedes buscar alimentos para obtener sus calorias automaticamente:

  • Escribe el nombre del alimento (minimo 2 caracteres) y pulsa Buscar
  • El sistema consulta la base de datos de OpenFoodFacts
  • Selecciona el alimento correcto de los resultados
  • Las kcal se calculan automaticamente segun la cantidad

Entrada por voz

En el editor de dieta manual, puedes dictar el nombre de los alimentos pulsando el icono del microfono junto al campo de texto. El sistema usa reconocimiento de voz del navegador.

Importar desde JSON

  1. Pulsa el boton Import en el dashboard
  2. Selecciona un archivo JSON previamente exportado
  3. La dieta se importa con toda su estructura (dias, comidas, alimentos)

Exportar dieta a JSON

Pulsa Export junto a cualquier dieta para descargarla en formato JSON. Util para hacer copias de seguridad o compartir dietas con otros usuarios.

Editar dieta existente

Pulsa Editar junto a cualquier dieta para abrir el editor manual con los datos actuales. Puedes modificar dias, comidas y alimentos.

Al editar una dieta, los planes semanales que ya se crearon a partir de ella no se modifican. Solo los nuevos planes usaran la version actualizada.

4. Plan semanal

El plan semanal es tu dieta organizada por dias (Lunes a Domingo) con todas las comidas de cada dia. Es donde haces el seguimiento diario.

Crear plan desde dieta

Pulsa el boton Plan junto a cualquier dieta en el dashboard. Se crea automaticamente un plan para la semana actual (empieza el lunes).

Crear plan manual

  1. Pulsa el boton Plan (naranja) en el dashboard
  2. Pon un nombre y selecciona la fecha de inicio
  3. Se crean 7 dias vacios que puedes rellenar manualmente

Navegar por los dias

Usa las pestanyas en la parte superior (Lun, Mar, Mie, Jue, Vie, Sab, Dom) para ver las comidas de cada dia.

Marcar comidas completadas

  • Pulsa el circulo junto a cada comida para marcarla como completada (aparece un check verde)
  • Vuelve a pulsar para desmarcarla
  • El porcentaje de cumplimiento se actualiza automaticamente en el dashboard

Completar dia entero

Pulsa el boton Completar dia para marcar todas las comidas del dia actual como completadas de golpe. Si ya estan todas completadas, las desmarca.

Anadir comidas al plan

  1. Pulsa + Anadir comida
  2. Selecciona el tipo de comida (Desayuno, Almuerzo, etc.)
  3. Escribe la descripcion (ej: "Pollo a la plancha 200g, arroz integral 150g")
  4. Pulsa Guardar

Eliminar comidas

Pulsa el boton X junto a cualquier comida para eliminarla del plan.

Cambiar entre planes

Si tienes varios planes, usa el desplegable en la parte superior de la pagina de Plan para seleccionar el plan que quieres ver.

Repetir plan

Si tu plan tiene una dieta asociada, aparece el boton Repetir. Al pulsarlo:

  • Si la semana del plan ya ha pasado: crea un nuevo plan para el proximo lunes desde hoy
  • Si la semana del plan aun no ha terminado: crea el plan para la semana siguiente
No se pueden crear dos planes para la misma semana. Si ya existe uno, el sistema te avisara.

Exportar plan a PDF

Pulsa Descargar PDF para generar un documento PDF con tu plan semanal completo, listo para imprimir.

Eliminar plan

Pulsa el boton Eliminar para borrar el plan completo (incluye todos los dias, comidas y lista de la compra asociada).

5. Lista de la compra

Se genera automaticamente a partir de tu plan semanal activo, agrupando todos los ingredientes que necesitas.

Generar lista

  • Ve a la seccion Compra en la barra de navegacion inferior
  • La lista se genera automaticamente del plan activo
  • Si no hay lista, pulsa Generar lista

Categorias

Los items se agrupan automaticamente por categoria: Proteinas, Carbohidratos, Grasas, Verduras, Frutas, Bebidas, Suplementos, Condimentos y Otros.

Cantidades semanales

  • Cada item muestra la cantidad total necesaria para toda la semana
  • Las cantidades iguales se suman automaticamente (ej: 7 x 100g = 700g)
  • Al final hay un resumen con el total de items por categoria

Marcar como comprado

Pulsa el circulo junto a cada item para marcarlo como comprado. Util para ir tachando en el supermercado.

Anadir item manual

Escribe el nombre y cantidad de un producto extra y pulsa + para anadirlo a la lista.

Eliminar item

Pulsa X junto a cualquier item para eliminarlo.

Regenerar lista

Pulsa Regenerar para recalcular la lista desde el plan actual. Los items anadidos manualmente se mantienen.

6. Control de peso

Lleva un seguimiento de tu peso a lo largo del tiempo para ver tu evolucion.

Registrar peso

  1. Ve a la seccion Peso
  2. Pulsa + Anadir
  3. Introduce tu peso en kg (acepta decimales, ej: 72.5)
  4. Selecciona la fecha (por defecto es hoy)
  5. Opcionalmente anade una nota (ej: "Despues de desayunar")

Ver historial

La lista muestra todos tus registros ordenados del mas reciente al mas antiguo, con:

  • Fecha y peso
  • Flechas de tendencia: subida, bajada o mantenimiento
  • Notas si las hay

Eliminar registro

Pulsa el boton de eliminar junto a cualquier registro para borrarlo.

Solo puedes registrar un peso por dia. Si intentas registrar otro el mismo dia, se actualizara el existente.

7. Perfil y seguridad

Datos personales

  • Usuario - Tu nombre de usuario (no se puede cambiar)
  • Email - Puedes actualizar tu email
  • Pulsa Guardar cambios para confirmar

Cambiar contrasenya

  1. Introduce tu contrasenya actual
  2. Introduce la nueva contrasenya (mismos requisitos que al registrarte)
  3. Pulsa Cambiar contrasenya

Autenticacion 2FA

Desde aqui puedes activar o desactivar la verificacion en dos pasos. Ver seccion Cuenta y acceso para mas detalles.

8. Panel de administracion Admin

Solo visible para usuarios con rol de administrador o superior.

Gestion de usuarios

Lista de todos los usuarios del sistema con acciones disponibles:

  • Crear usuario - Nombre, email, contrasenya y rol
  • Editar - Modificar email, rol y estado activo/inactivo
  • Reset pass - Establecer nueva contrasenya para un usuario
  • Impersonar - Entrar como otro usuario para ver su plan y datos (util para soporte)
  • Eliminar - Borrar usuario y todos sus datos
Al impersonar un usuario, tu sesion se reemplaza por la del usuario seleccionado. Para volver, cierra sesion y entra con tu cuenta de admin.

Roles disponibles

  • User Estandar - Acceso a su propio plan, dietas, peso y compra
  • Superuser Admin - Acceso a funciones avanzadas y panel de admin
  • SuperUserMaster Master - Acceso total: crear/editar/eliminar usuarios, base de datos, impersonar

9. Base de datos SuperUserMaster

Herramientas avanzadas solo disponibles para SuperUserMaster.

Explorador de tablas

  • Ver todas las tablas de la base de datos
  • Navegar por los registros con paginacion
  • Ver columnas con tipos de datos e indicador de clave primaria

Consola SQL

10. Modo offline y PWA

EatHealthyCycle funciona como una aplicacion instalable (PWA) y tiene soporte offline limitado.

Instalar como app

En tu navegador movil o de escritorio, busca la opcion "Instalar aplicacion" o "Anadir a pantalla de inicio". La app se abrira como una aplicacion nativa sin barra de navegacion del navegador.

Funcionamiento offline

  • Los datos que ya hayas cargado (dietas, planes, lista de compra, peso) se guardan en cache y estan disponibles sin conexion
  • Puedes marcar comidas como completadas y tachar items de la compra sin conexion
  • Las acciones realizadas offline se guardan en una cola de sincronizacion
  • Cuando vuelva la conexion, se sincronizan automaticamente con el servidor

Indicador offline

Cuando no hay conexion, aparece una barra naranja en la parte superior indicando el modo offline y el numero de acciones pendientes de sincronizar.

11. Portal de aplicaciones

Pulsa el boton Apps en la esquina superior derecha del dashboard para acceder al portal de aplicaciones. Desde ahi puedes navegar entre:

  • EatHealthyCycle - Esta aplicacion de dietas y nutricion
  • FitCycle - Aplicacion de entrenamiento y ejercicio

12. Actualizaciones

Cuando se despliega una nueva version de la aplicacion:

  • Aparece automaticamente un popup informando de la nueva version
  • Pulsa Actualizar ahora para recargar con la ultima version
  • Pulsa Mas tarde para continuar con la version actual

La version actual se muestra debajo del titulo EatHealthyCycle en el dashboard (ej: v1.1.2).

El indicador (P) significa Production y (D) significa Develop.

13. Tipos de comida

La aplicacion soporta 7 tipos de comida, cada uno con su color identificativo:

  • Pre Desayuno - Antes de entrenar o primera hora
  • Desayuno
  • Media Manana - Snack de media manana
  • Almuerzo
  • Comida - Comida principal del mediodia
  • Merienda
  • Cena
